Registration Form

Guidelines

Use 25+ number to change the state code, starting from 100.
Eg. 100, 125, 150, 175, 200, 225, etc. Numbers in these formats are valide. All other numbers are invalid.

After submitting this form you will be redirected to the OTP validation page and the new window will open. Just becouse I have not used a web API here, you will have to select the Form page's window from window tab of your browser where you will find the system generated OTP. After following this you will be able copy the OTP and Paste it to the OTP field of newly opened window.
